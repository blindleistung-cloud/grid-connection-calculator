# **App Name**: PV Connection Scenario Comparator

## Core Features:

- Technical Input & Validation: Allow users to input PV generated power, distance, maximum voltage drop, conductor material, and other technical parameters. Validate inputs for sensible ranges and numeric values.
- Cable Sizing Calculation: Calculate the required current and estimate the minimum cable cross-section for both Low-Voltage (LV) and Medium-Voltage (MV) scenarios, based on a simplified voltage-drop model.
- Cost Assumption Input: Enable users to enter project-specific cost assumptions for LV/MV cables, installation, transformer, and MV station components.
- Economic Scenario Comparison: Automatically compare the total estimated costs for the LV and MV scenarios, calculate absolute and relative cost differences, and provide a recommendation.
- Results Display: Present a clear summary of technical sizing details, economic comparison, and an engineering comment on the scenario findings for easy review and potential copy/paste.

## Style Guidelines:

- Primary color: #3388CC (a balanced, professional blue) to convey trust, clarity, and precision for an engineering tool.
- Background color: #EDF3F6 (a very light, desaturated blue) to maintain a clean, readable canvas, supporting the light color scheme.
- Accent color: #42CFBB (a vibrant aqua-green) analogous to the primary, providing visual emphasis for key actions or highlighted information without being distracting.
- Headline and body font: 'Inter' (a sans-serif) for its modern, neutral, and highly legible characteristics, suitable for displaying technical data and fostering clear understanding.
- Utilize simple, outline-style icons that are easily recognizable and convey technical concepts clearly without clutter, maintaining a professional appearance.
- Employ a structured, potentially grid-based layout for input forms and result tables to ensure clear presentation of numerical data and guide the user through the two-step workflow (technical input, commercial input).
- Incorporate subtle, functional animations for feedback on input validation, calculation processes, and transitions between workflow steps, enhancing the user experience without introducing unnecessary visual distractions.
